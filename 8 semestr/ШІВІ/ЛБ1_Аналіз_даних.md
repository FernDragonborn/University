#### 3.6. Завдання 7. Побудова графіків для аналізу даних

**Порівняння 3 стовпців по містах:**

```python
import matplotlib.pyplot as plt
from matplotlib.axes import Axes
from typing import Tuple

df = pd.read_csv('city_happiness_train.csv')
top_cities = df.groupby('City').agg({
    'Happiness_Score': 'mean',
    'Air_Quality_Index': 'mean',
    'Healthcare_Index': 'mean'
}).sort_values('Happiness_Score', ascending=False).head(15)

fig, axes = plt.subplots(1, 3, figsize=(18, 8), dpi=100)
axes: Tuple[Axes, Axes, Axes] = axes

axes[0].barh(top_cities.index, top_cities['Happiness_Score'], color='steelblue')
axes[0].set_title('Happiness Score')
axes[0].invert_yaxis()
axes[0].set_xlim(7.5, 8.6)

axes[1].barh(top_cities.index, top_cities['Air_Quality_Index'], color='coral')
axes[1].set_title('Air Quality Index')
axes[1].invert_yaxis()
axes[1].set_xlim(17, 42)

axes[2].barh(top_cities.index, top_cities['Healthcare_Index'], color='mediumseagreen')
axes[2].set_title('Healthcare Index')
axes[2].invert_yaxis()
axes[2].set_xlim(89, 100)

plt.suptitle('Порівняння показників топ-15 міст (за Happiness Score)')
plt.tight_layout()
plt.savefig('comparison_3cols.png', dpi=150)
plt.show()
```
