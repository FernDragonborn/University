#### 3.1. Завдання 1–2. Імпорт бібліотек, завантаження та підготовка даних

Оскільки ми працюємо з реальними даними, необхідно обробити пропущені значення (NaN) та звести всі ознаки до єдиного масштабу від 0 до 1 за допомогою `MinMaxScaler`, як це розглядалося в лекційних матеріалах.

```python
from typing import cast
import pandas as pd
from sklearn.impute import SimpleImputer
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import MinMaxScaler

path = "world_happiness_report_2005_2025.csv"
df: pd.DataFrame = cast(pd.DataFrame, pd.read_csv(path))
df.columns = df.columns.str.strip()
feature_cols = [
    'year',
    'explained_log_gdp_per_capita',
    'explained_social_support',
    'explained_healthy_life_expectancy',
    'explained_freedom',
    'explained_generosity',
    'explained_corruption'
]
target_col = 'happiness_score'

X = df[feature_cols]
y = df[target_col]

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

imputer = SimpleImputer(strategy='median')
X_train_imp = imputer.fit_transform(X_train)
X_test_imp = imputer.transform(X_test)
y_train = y_train.fillna(y_train.median())
y_test = y_test.fillna(y_test.median())

scaler = MinMaxScaler()
X_train_scaled = scaler.fit_transform(X_train_imp)
X_test_scaled = scaler.transform(X_test_imp)
```

#### 3.2. Завдання 3. Навчити `MLPRegressor` та `Ridge`, порівняти їхню якість за метрикою R² та MSE.

Для нейронної мережі створюємо два прихованих шари (наприклад, 100 та 50 нейронів) і збільшуємо максимальну кількість ітерацій `max_iter`, щоб модель встигла зійтися.

```python
from sklearn.neural_network import MLPRegressor
from sklearn.linear_model import Ridge
from sklearn.metrics import r2_score, mean_squared_error
mlp_model = MLPRegressor(
    hidden_layer_sizes=(100, 50),
    activation='relu',
    solver='adam',
    max_iter=1000,
    random_state=42
)
mlp_model.fit(X_train_scaled, y_train)

ridge_model = Ridge(alpha=1.0)
ridge_model.fit(X_train_scaled, y_train)

y_pred_mlp = mlp_model.predict(X_test_scaled)
y_pred_ridge = ridge_model.predict(X_test_scaled)

r2_mlp = r2_score(y_test, y_pred_mlp)
mse_mlp = mean_squared_error(y_test, y_pred_mlp)

r2_ridge = r2_score(y_test, y_pred_ridge)
mse_ridge = mean_squared_error(y_test, y_pred_ridge)

print(f"MLPRegressor -> R^2: {r2_mlp:.4f}, MSE: {mse_mlp:.4f}")
print(f"Ridge        -> R^2: {r2_ridge:.4f}, MSE: {mse_ridge:.4f}")
```

fig, axes = plt.subplots(1, 2, figsize=(14, 6), sharey=True)
